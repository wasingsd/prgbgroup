# PR Global Group Website

Official website for **PR Global Group**, a comprehensive China-Thailand logistics and travel service provider.

## Features
- **PR CARGO**: Import/Export service information, process steps, and pricing requests.
- **PR TRAVEL**: Tour packages, business trips, and unseen china routes.
- **Responsive Design**: Modern UI built with Tailwind CSS.
- **Contact Form**: Integrated contact options.

## Technology Stack
- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Font**: Prompt (Google Fonts)

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Deployment to Production

The easiest way to deploy this Next.js app is to use [Vercel](https://vercel.com):

1.  Create a Vercel account.
2.  Import your GitHub repository (`prgrbgroup`).
3.  Vercel will automatically build and deploy your site.
